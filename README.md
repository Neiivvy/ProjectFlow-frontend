# Team Project Management and Progress Tracking System (Frontend)

##  Project Overview
This is the frontend of a Team Project Management and Progress Tracking System built using Next.js and Tailwind CSS.  

The system helps teams manage projects, assign tasks, track progress, and improve collaboration between members, project managers, and admins.

---

##  Tech Stack
- Next.js
- React
- Tailwind CSS
